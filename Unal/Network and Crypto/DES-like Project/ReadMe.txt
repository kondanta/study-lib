## Source Code:
- main.py ==> Main library of the code. 
- encryption.py ==> Use for encryption of "input.txt" file.
- decryption.py ==> Use for decryption of "encrypted.txt" which is created by encryption.py.

### Used Libraries:
- os ==> Used for os.remove and os.system("pause")

## Steps:
- Run encryption.py. This will encrypt the content of "input.txt" and will create "encrypted.txt" as output.
- Run decryption.py. This will decrypt the content of "encrypted.txt" and will create "decrypted.txt" as output.

!! Note: While testing please check if the character you are using in the input.txt is exist in Encode Table in main.py.
